# Local Search Engine Project

This is a C++ search engine project built from scratch using SQLite for data storage. It is designed to be fully standalone and portable.

## Prerequisites
- A Windows computer
- **g++** compiler installed and added to your `PATH` (e.g., via MinGW).

## How to Build and Run
All necessary dependencies (SQLite header, static library, and runtime DLL) are bundled directly inside this repository.

1. Open your terminal or command prompt in the project directory.
2. Run the provided batch script to compile and execute the program:
   ```cmd
   .\run.bat
   ```

3. When prompted, enter the directory you wish to traverse and index.

Alternatively, you can just run the pre-compiled executable directly without rebuilding:
```cmd
.\search_engine.exe
```
